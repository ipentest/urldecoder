URL DEncoder by @iPenTest

  • Input a string of text and encode or decode it as you like.
  • Alphanumerics are not touched... Only the reserved characters in a URL are encoded
  • Go to https://github.com/ipentest/urldecoder/blob/master/index.html Save as HTML and you're good to go offline
